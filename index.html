Tecnologias

HTML5

Meu primeiro contato foi com o HTML (HyperText Markup Language), que é a base de qualquer site. Ele define a estrutura da página, organizando o conteúdo em elementos como títulos, parágrafos, imagens, links e listas.

CSS3

Depois de aprender HTML, comecei a estudar CSS (Cascading Style Sheets), que é responsável pelo design e pela aparência do site. No início, mexer com estilos parecia simples, mas logo percebi que organizar um layout bem feito pode ser desafiador.

Git

Além das linguagens de programação, também comecei a aprender sobre Git, uma ferramenta essencial para controle de versão. Com Git, posso salvar e gerenciar diferentes versões dos meus projetos, o que me permite testar mudanças sem medo de perder o que já fiz.

JavaScript

Depois de me sentir mais confortável com HTML e CSS, comecei a explorar o JavaScript, que é o que torna os sites interativos. Diferente do HTML e do CSS, que são mais visuais, o JavaScript envolve lógica de programação, algo que exige mais raciocínio

Figman

Figma é uma ferramenta de design de interface baseada na nuvem, usada para criar protótipos de sitesaplicativos. Permite colaboração em tempo real, facilitando o trabalho em equipe no desenvolvimento de interfaces visuais.

bootstrap

é um framework front-end que agiliza a criação de sites responsivos. Ele oferece um sistema de grid, componentes prontos e estilos baseados em CSS e JavaScript, tornando o desenvolvimento mais rápido e padronizado.
